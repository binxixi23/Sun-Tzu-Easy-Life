**Sun Tzu_Easy_Life** is a high-complexity, multi-agent AI framework designed to navigate the modern world using ancient strategic wisdom. It transforms the 13 Chapters of the *Art of War* into a digital "Sovereign Advisor" capable of solving problems across 25+ human domains.

## 🚀 The Digital Military Academy Features

- **Multi-Agent Orchestration:** A staff of specialized agents (Scout, Strategist, Visualizer, Banter) working in synchronization.
- **Big Data Intelligence:** Built-in armory of **1,000+ modern situational scenarios** across Politics, Wealth, and Communication.
- **Strategic Memory:** A living `battle_history.json` that learns from every interaction, ensuring the General never makes the same mistake twice.
- **Real-Time Reconnaissance:** Integrated **Web Scraper** that pulls live news to assess the "Climate" of your strategy.
- **Visual Command:** Automated **Mermaid.js** battle maps for every victory plan.

## 📁 Repository Structure

- `agents/`: The "General Staff" (Logic, Intelligence, Graphics, Humor).
- `core/`: The "Deep Wisdom" manifestos covering Politics, Wealth, and Human Nature.
- `data/`: The "Armory" (1,000+ Case Studies, Tactical Library).
- `scripts/tools/`: Advanced utilities for Web Scraping, Memory, and Calendar Sync.

## 🛠️ Installation & Deployment

1. **Clone the Fortress:**
   ```bash
   git clone https://github.com
   cd Sun-Tzu-Easy-Life
   ```

2. **Supply the Troops:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Consult the General:**
   ```bash
   python main.py
   ```
