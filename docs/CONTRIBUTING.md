# Contributing to Sun Tzu_Easy_Life

We welcome new "Scouts" and "Generals" to improve our library.

### How to Contribute:
1. **Add to the Library:** Submit new tactical hacks to `data/tactical_library.md`.
2. **New Scenarios:** Add to the 1,000+ modern scenarios in `data/case_studies.json`.
3. **Agent Logic:** Improve the `StrategistAgent`'s decision-making algorithm.

**Philosophy:** All contributions must aim to make life *easier* through *smarter* positioning.
